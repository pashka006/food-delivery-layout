# 🍔 Багатосторінковий сервіс доставки їжі

Адаптивна верстка сервісу доставки їжі, виконана на основі професійного макета з Figma. Проєкт включає понад 200 медіафайлів та декілька функціональних сторінок.

## 🔗 Посилання
* **Live Demo:** [Відкрити сайт](https://pashka006.github.io/food-delivery-layout/)
* **Дизайн:** [Макет у Figma](https://www.figma.com/design/uCFaTqHDtqVazzwgqVQs02/Food-Delivery?node-id=0-1&t=6uhAcFrzBsEfUosQ-1)
## 🛠 Стек технологій
* **HTML5:** Семантична розмітка.
* **CSS3:** Flexbox, Grid, адаптивність під мобільні пристрої.
* **Структура:** Використання відносних шляхів для стабільної роботи на хостингу.

## 📁 Особливості проєкту
- Повна адаптивність (mobile-first / responsive).
- Багатосторінкова структура (Головна, Ресторани, Кошик, Профіль).
- Коректна робота всіх внутрішніх посилань.
