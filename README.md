# 🍔 Food Delivery: Повний цикл розробки (Design & Frontend)

Це авторський проєкт, у якому я поєднав ролі UI/UX дизайнера та Frontend-розробника. Весь цикл — від створення візуальної концепції в Figma до написання чистого коду — виконано мною.

## 🔗 Посилання
* **Live Demo:** [Відкрити готовий сайт](https://pashka006.github.io/food-delivery-layout/)
* **Мій дизайн у Figma:** [Переглянути макет](https://www.figma.com/design/uCFaTqHDtqVazzwgqVQs02/Food-Delivery?node-id=0-1&t=6uhAcFrzBsEfUosQ-1)

## 🎨 Етап дизайну (UI/UX)
Як автор макета, я пропрацював:
- Колірну палітру та типографіку для кращого сприйняття сервісу їжі.
- Логіку переходів між сторінками (User Flow).
- Адаптивні сітки для мобільних пристроїв та десктопу.

## 💻 Етап розробки (Frontend)
- **Технології:** HTML5, CSS3 (Flexbox, Grid).
- **Обсяг:** Реалізовано багатосторінкову структуру та підключено понад 200 медіафайлів.
- **Особливість:** Код повністю відповідає макету (Pixel Perfect) та має адаптивну верстку.
